# Selenium Samples

Web automation and testing examples using Selenium WebDriver.

## Contents

### Jupyter Notebooks

Interactive notebooks demonstrating Selenium usage patterns:
- `markwarner.ipynb` - Selenium automation examples

## Usage

### Running Notebooks

1. Install Jupyter and dependencies:
```bash
pip install jupyter selenium
```

2. Start Jupyter:
```bash
jupyter notebook
```

3. Open and run the notebooks in your browser

### WebDriver Setup

Ensure you have the appropriate WebDriver installed:
- **Chrome:** ChromeDriver
- **Firefox:** GeckoDriver
- **Edge:** EdgeDriver
